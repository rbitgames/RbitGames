Quick Start

Get started quickly with the 2D fort building system

Import​

  1. Open the Package Manager of the unity editor

  1. Select My Assets

  1. Import the 2D fort build system

Demo Scenes​

There are three demo scenes included

Main Scene​

The Instruction scene has how-to instructions

Example1 Scene​

The Example1 scene demonstrate the basic functions of a 2D fort

Example2 Scene​

The Example2 scene also demonstrates the basic functionality of a 2D fort, but with a changed background and the fort is built upside down

From Scratch​

The BuildFromZero scene Build a fort from scratch and you can save forts that have already been built

How to save the fort​

Built a fort from scratch. save it, it will remain the same for the next time play the game.
This is the fort I have already built, it looks like this.

  1. Then right-clicked on the Fort game object in the hierarchy window and selected Copy.
    Also make sure that the scale of the transform component values is zero.

  1. Exit play.

  1. Right-clicked in the hierarchy window and selected Paste.

  1. After pasted it looks like this.

  1. Delete the previous Fort

  1. Hit Play, it will be the same as what you built before.
