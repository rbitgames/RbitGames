Wood

Wood prefab explanation.

Hierarchy​

Wood​

The Wood containing component is displayed in the Inspector, Body Type of Rigidbody2d set to Kinematic. view Stick component

StickSprite​

The StickSprite containing component is displayed in the Inspector. set wood texture to sprite of sprite renderer.

EndPoint​

The EndPoint just containing a Transform component, and set the position to the top of the wood.

CenterPoint​

The CenterPoint just containing a Transform component, and set the position to the center of the wood.

WeaponPoint​

The WeaponPoint just containing a Transform component, and set the position above the center of the wood.

WeaponPoint_upsideDown​

The WeaponPoint_upsideDown just containing a Transform component, and set the position below the center of the wood.
