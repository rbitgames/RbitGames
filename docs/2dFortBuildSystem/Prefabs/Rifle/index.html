Rifle

Rifle prefab explanation.

Hierarchy​

Rifle​

The Rifle includes the following components, view Weapon component.

Base / Gun​

The Base and Gun game objects only have one sprite renderer component. the Gun game object is rotatable part.

MuzzleFlash​

This is just a particle, the muzzle spark effect during shooting.

AttackZone​

The AttackZone contains an Attackzone component, which references child objects of the AttackZone.

Change attackzone's length​

  1. Select UpperLine and LowerLine.

  1. Changing the position of the Line Renderer component will change the length of the attack area.
